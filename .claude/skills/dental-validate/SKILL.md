# 포스팅 검증 체크리스트

## 자동 검증 항목 (post-validator.ts)

| # | 체크 | 기준 |
|---|------|------|
| 1 | 글자수 | 2,000자+ (공백 제외), 2,500자+ 권장 |
| 2 | 치과명 위치 | 서문 인사 + 마무리 인사에만 허용 |
| 3 | 금지 어미 | ~해요/~거든요/~있어요 등 0건 |
| 4 | 키워드 빈도 | 형태소A/B 각 7회 |
| 5 | 의료법 | 효과 보장/약속 표현 0건 |
| 6 | 금칙어 | 독립 단어 기준 0건 |
| 7 | 부작용 고지 | 시술글 필수 |
| 8 | 동의어 회전 | 전체 6회 초과 단어 0건 |
| 9 | 어미 다양성 | ~입니다 3연속 이하 (정보성) |
| 10 | AI 패턴 | 번호 목록/상투어 0건 |
| 11 | 논문 인용 | citePapers 시 [1] + References 존재 |

## 글자수 제외 항목
- 📷 이미지 플레이스홀더
- 📎 논문 인용 블록 (References)
- 부작용 고지문 (※ ...)
- 출처 표기 ((출처: ...))

## 예외 규칙 (화이트리스트)

### 필수 허용 표현
- **존댓말 어미**: ~하세요, ~드세요, ~으세요, ~시죠, ~하시면
- **문법 요소**: ~하지 않다, ~되지 않다, ~적인, ~적으로, ~등의, ~에 대해, ~위해, ~때문에
- **치과 전문 용어**: 신경치료, 신경관, 치신경, 임플란트 식립, 보철물 삽입, 치주, 치은, 치아, 치료, 치과, 교정치료, 교정장치, 발치, 발생, 예방

### 검사 우선순위
1. 독립 단어로 사용된 금칙어 → 반드시 대체
2. 복합어 내 금칙어 → 전체 단어가 자연스러우면 허용
3. 전문용어 내 금칙어 → 화이트리스트 확인 후 허용

### 대체 예시
- 걱정 → 염려, 우려
- 고민 → 생각, 숙고
- 고통 → 아픔, 통증
- 너무 → 매우, 굉장히
- 해결 → 처리, 극복
- 불안 → 초조, 마음이 편치 않은
