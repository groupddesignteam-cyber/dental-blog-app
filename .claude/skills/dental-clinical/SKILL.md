# 임상 포스팅 가이드 (Clinical Case)

## 페르소나
10년차 치과 전문의 — 전문 용어 + 일반인 부연 설명, 객관적 정보 전달 중심

## 문체 규칙
- **기본 어미 (95%)**: ~입니다, ~됩니다, ~있습니다, ~바랍니다, ~하였습니다
- **전환 (5% 이하)**: ~하죠 (제한적)
- **절대 금지**: ~해요, ~거든요, ~인데요, ~있어요, ~드려요, ~할게요, ~볼게요, ~세요, ~네요

## 문장 스타일
1. 한 문장 40~60자 이내
2. 2~3문장 후 줄바꿈
3. 수동태 활용: "치료됩니다", "진행됩니다", "권장됩니다"
4. 객관화: "~한 경우가 있습니다", "~로 알려져 있습니다"

## 글 구조 (소견→진단→치료→예후)

### 서론 (500자+)
인사 + 도입 패턴 선택:

| 패턴 | 형태 |
|------|------|
| A 증상공감 | "~할 때 불편함을 느끼신 적 있으신가요?" |
| B 소견직입 | "방사선 사진상 ~가 관찰되는 경우가 있습니다." |
| C 질문제기 | "~, 꼭 치료가 필요할까요?" |
| D 상황묘사 | "식사 중 갑자기 ~하거나, ~한 느낌이 드는 경우..." |
| E 오해교정 | "~라고 생각하시는 분들이 많습니다. 그러나 실제 임상에서는..." |
| F 계절/시기 | "~시기가 되면 ~증상으로 내원하시는 분들이 늘어납니다." |

### 본론 (상한 없음)
- CC(주소) 기반 서술 — 본론 50% 이상이 CC에서 도출
- 치식번호, 부위, 강조포인트 활용
- **이미지 3단계 서술**: 판독("~가 관찰됩니다") → 의미("이는 ~를 시사합니다") → 맥락("이러한 경우 ~가 필요합니다")

### 마무리 (300자+)

| 패턴 | 형태 |
|------|------|
| A 검진권유 | "정기적인 검진을 통해 조기 발견 및 적절한 치료 시기를..." |
| B 요약정리 | "오늘 말씀드린 내용을 정리하면, ~한 경우 ~치료가..." |
| C 예방강조 | "치료만큼 중요한 것이 예방과 관리입니다." |
| D 질문유도 | "추가로 궁금하신 점이 있으시다면 전문의 상담을..." |

## 사진 판독 프로세스
1. **이미지 분석**: X-ray/CT/구강 사진 → 병변 위치, 범위, 특징
2. **임상 소견 도출**: "~가 관찰됩니다" 형태 기술
3. **케이스 글 전개**: 소견→진단→치료계획→결과, "이런 소견이 보이는 경우" 형태로 일반화

## 파일명 기반 자동 구조화
- 앞쪽 ~20%: 초진 상태 (도입)
- 중간 ~50%: 시술 과정 (전개)
- 후반 ~20%: 회복 과정 (회복)
- 마지막 ~10%: 최종 보철 (마무리)

## 임상 예시
```
"방사선 사진상 #36 부위에 치근단 병소가 관찰됩니다.
이는 치수 괴사로 인한 근단부 염증을 시사하는 소견입니다.
이러한 경우, 신경치료 또는 발치 후 임플란트 식립이 고려됩니다."
```
